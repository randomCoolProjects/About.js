About.js

About.js é uma bibloteca (Javascript) para obter dados do usuário, tais como: Navegador, Sistema Operacional, endereço de IP, localização (via ip), etc...
É suportada pela maioria dos navegadores e é simples de usar:
var about = About();
window.setTimeout(function(){
console.log(about);
}, 2000); // 2 segundos de tempo para receber os dados
